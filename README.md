# Android Ultima Tirada

App Android nativa en Kotlin y Jetpack Compose basada en la app SwiftUI de `AppUltimaTirada`.

Incluye:

- Sesión con token contra `https://ultimatirada.com`.
- Home con estadísticas, eventos, productos destacados y ranking.
- Tienda con buscador, filtros por marca y carrito local persistente.
- Formulario de venta de cartas conectado a la API.
- Eventos con inscripción básica.
- Comunidad con publicaciones, likes y feed.
- Perfil con datos del usuario y ranking.

## Abrir

Abre esta carpeta con Android Studio y sincroniza Gradle. La app usa el módulo `app`.
